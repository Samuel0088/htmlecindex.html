    C#

    Console.Write= Escreve algo na mesma linha

    Console.WriteLine= Escreve algo na linha de baixo

    int= Usado para se referir a números

    string= Usado para se referir a letras

    if= se

    else= Contrario do if

    {}= As chaves são usadas para 2 ou mais funções no if ou else

    Console.ReadKey= Mantém a aba aberta mesmo quando uma tecla for pressionada

    Laços de Repetição

    For

    for (int i=0; i < args.Lenght; i++)

    i=0 é o valor inicial

    < args.Lenght é o critério de parada

    i++ é o incremento

    While

    while= Significa enquanto

    while (contador <= a variavel)
    {
    Console.WriteLine
    contador++

    Console.WriteLine é o contador e contador++ é o incremento

    Do...While

    do
    {
    Console.WriteLine(primeira variavel);
    primeira variavel++
    }While (primeira variavel <= segunda variavel);

    Sem o incremento ele fica em loop infinito
